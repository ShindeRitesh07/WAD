Step 1: Login to AWS Console and open EC2 service.

Step 2: Click Launch Instance.

Step 3: Enter instance name: WebServer.

Step 4: Select Amazon Linux 2023 AMI.

Step 5: Choose instance type t2.micro.

Step 6: Create and download a new key pair (.pem file).

Step 7: Allow SSH (22) and HTTP (80) traffic in security group.

Step 8: Click Launch Instance.

Step 9: Wait until instance state becomes Running.

Step 10: Connect using EC2 Instance Connect.

Step 11: Run commands:
sudo yum update -y
sudo yum install httpd -y
sudo systemctl start httpd
sudo systemctl enable httpd

Step 12: Create webpage:
cd /var/www/html
sudo nano index.html

Step 13: Add HTML content:
<h1>Welcome to AWS Apache Server</h1>

Step 14: Open browser and type: http://PUBLIC_IP
